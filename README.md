# 🐋 Whale Jumper

A simple 2D platformer built with [SvelteKit](https://svelte.dev/docs/kit). Play as the **DeepSeek whale**, collect all the coins, and jump your way to victory!

**[▶ Play now](https://riziles.github.io/whale-jumper/)**

## Controls

| Key | Action |
|---|---|
| Arrow keys / WASD | Move left & right |
| Space / Up / W | Jump |

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:5173/whale-jumper](http://localhost:5173/whale-jumper) in your browser.

## Build

```bash
pnpm build
```

Static output lands in `build/`.
